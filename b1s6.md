### Ejercicio 1
Implementa una función `findEmployeeSalary` que reciba el nombre de un empleado y devuelva su salario como un `Option[Double]`. Si el empleado no existe en la lista, la función debe retornar `None`.

```Scala
val employees = List(
  ("Alice", 30, "IT", 5000.00, "New York"),
  ("Bob", 45, "HR", 3000.50, "Los Angeles"),
  ("Charlie", 25, "Finance", 4500.75, "Chicago")
)



def encontrarSalariosEmpleados(name: String)={
  employees.find(_._1==name).map(e=>(e._4,e._1))
}

println(encontrarSalariosEmpleados("Alice"))
println(encontrarSalariosEmpleados("Bob"))
println(encontrarSalariosEmpleados("Charli"))
```


### Ejercicio 2
Implementa una función safeSqrt que reciba un número Double y devuelva un Try[Double]. Si el número es negativo, la función debe manejarlo con un Failure, indicando que no se puede calcular la raíz cuadrada de números negativos.

```Scala
import scala.util.{Try, Success, Failure}

val employees = List(
  ("Alice", 30, "IT", 5000.00, "New York"),
  ("Bob", 45, "HR", 3000.50, "Los Angeles"),
  ("Charlie", 25, "Finance", 4500.75, "Chicago")
)

// Función para encontrar el salario de un empleado dado su nombre
def findEmployeeSalary(name: String) = {
  employees.find(_._1 == name).map(e => (e._1, e._4))
}

// Ejemplos de uso
println(findEmployeeSalary("Alice"))
println(findEmployeeSalary("Bob"))
println(findEmployeeSalary("David"))
```
